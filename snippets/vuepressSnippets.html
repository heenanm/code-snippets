# Frontmatter configuration

---
title: Vuepress
---

To make changes to page titles e.t.c the frontmatter yaml file can be configured using the above snippet. This must be placed at the top of the markdown file is applies to. If this isn't configured then the first H1 on the page will be used by default.

# Angle Brackets in Markdown

When adding angle brackets to in the markdown file either use a space e.g < my-text > or use the html codes '&lt;' and '&gt;'. If you dont do this then the text will not render to the page correctly as vuepress will see it as a component.
