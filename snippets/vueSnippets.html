# Displaying Data

# Interpolation

Double curly braces are used for interpolation

<p>{{ myData.prop }}</p>

# Event Handling

The @ symbol can be used as a shortcut for v-on:event

# Binding to a property

:property can be used as a shortcut for v-bind:property

# 2 Way Binding

v-model="property" is used to make the binding 2 way for example when used with an input The current value will be shown but if the user sets the input the data model will change to the set value.

# Conditionally add/remove content from the DOM

v-if="condition" is used to conditionally add content to the DOM.

<div v-if="showContent">
    <p> div is added if true </p> 
</div>

The content isn't rendered to the DOM if the condition is false.

# Conditionally show/hide content in the DOM

v-show="condition" can be used to add display: hidden CSS property to content in the DOM.
This leaves the content in the DOM and hides it from the user it will still be visable in developer tools.

# Iterate over data

v-for="item in itemList" can be used to iterate over a list/array of items and repeat a template. best practice is to combine this with :key="item.itemId" by binding a key to a unique identifier vue can be more efficient and only change data when necessary rather than doing a full iteration over the data each time.
